#define REMAINDER (   A,

)    (((A) + (B) * 1024) % (B))

Used for GL_REPEAT wrap mode.

Using A % B doesn't produce the right results for A<0. Casting to A to be unsigned only works if B is a power of two. Adding a bias to A (which is a multiple of B) avoids the problems with A < 0 (for reasonable A) without using a conditional.
