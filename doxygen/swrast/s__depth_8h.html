void _swrast_depth_clamp_span ( struct gl_context *  ctx,
SWspan *  span 
)

Clamp fragment Z values to the depth near/far range (glDepthRange()).

This is used when GL_ARB_depth_clamp/GL_DEPTH_CLAMP is turned on. In that case, vertexes are not clipped against the near/far planes so rasterization will produce fragment Z values outside the usual [0,1] range.
