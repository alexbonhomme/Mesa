Detailed Description

Remap table management.

Entries in the dispatch table are either static or dynamic. The dispatch table is shared by mesa core and glapi. When they are built separately, it is possible that a static entry in mesa core is dynamic, or assigned a different static offset, in glapi. The remap table is in charge of mapping a static entry in mesa core to a dynamic entry, or the corresponding static entry, in glapi.
