Detailed Description

Color, depth, stencil packing functions.

Helper struct for MESA_FORMAT_Z32_FLOAT_X24S8.

Used to pack basic color, depth and stencil formats to specific hardware formats.

There are both per-pixel and per-row packing functions:

  • The former will be used by swrast to write values to the color, depth, stencil buffers when drawing points, lines and masked spans.
  • The later will be used for image-oriented functions like glDrawPixels, glAccum, and glTexImage.Helper struct for MESA_FORMAT_Z32_FLOAT_X24S8
