void _mesa_override_gl_version ( struct gl_context *  ctx)

Override the context's version and/or API type if the environment variable MESA_GL_VERSION_OVERRIDE is set.

Example uses of MESA_GL_VERSION_OVERRIDE:

2.1: select a compatibility (non-Core) profile with GL version 2.1 3.0: select a compatibility (non-Core) profile with GL version 3.0 3.0FC: select a Core+Forward Compatible profile with GL version 3.0 3.1: select a Core profile with GL version 3.1 3.1FC: select a Core+Forward Compatible profile with GL version 3.1
