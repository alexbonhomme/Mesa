unsigned gl_transform_feedback_object::GlesRemainingPrims

GLES: if Active is true, remaining number of primitives which can be rendered without overflow.

This is necessary to track because GLES requires us to generate INVALID_OPERATION if a call to glDrawArrays or glDrawArraysInstanced would overflow transform feedback buffers. Undefined if Active is false.

Not tracked for desktop GL since it's unnecessary.

Referenced by _mesa_BeginTransformFeedback(), _mesa_validate_DrawArrays(), and _mesa_validate_DrawArraysInstanced().
