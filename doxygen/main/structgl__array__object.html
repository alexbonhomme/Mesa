Defined by the GL_APPLE_vertex_array_object extension, but a nice encapsulation in any case.

GLboolean gl_array_object::ARBsemantics

Does the VAO use ARB semantics or Apple semantics?

There are several ways in which ARB_vertex_array_object and APPLE_vertex_array_object VAOs have differing semantics. At the very least, 

- ARB VAOs require that all array data be sourced from vertex buffer
  objects, but Apple VAOs do not.

- ARB VAOs require that names come from GenVertexArrays.

This flag notes which behavior governs this VAO.
