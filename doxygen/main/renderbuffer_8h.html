void _mesa_reference_renderbuffer_ ( struct gl_renderbuffer **  ptr,
struct gl_renderbuffer *  rb 
)

Set *ptr to point to rb.

If *ptr points to another renderbuffer, dereference that buffer first. The new renderbuffer's refcount will be incremented. The old renderbuffer's refcount will be decremented. This is normally only called from the _mesa_reference_renderbuffer() macro when there's a real pointer change.
