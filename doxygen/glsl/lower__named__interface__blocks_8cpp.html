Detailed Description

This lowering pass converts all interface blocks with instance names into interface blocks without an instance name.

For example, the following shader:

out block { float block_var; } inst_name;

main() { inst_name.block_var = 0.0; }

Is rewritten to:

out block { float block_var; };

main() { block_var = 0.0; }

This takes place after the shader code has already been verified with the interface name in place.

The linking phase will use the interface block name rather than the interface's instance name when linking interfaces.

This modification to the ir allows our currently existing dead code elimination to work with interface blocks without changes.
