Calls a user function for every basic block in the instruction stream.

Basic block analysis is pretty easy in our IR thanks to the lack of unstructured control flow. We've got:

ir_loop (for () {}, while () {}, do {} while ()) ir_loop_jump ( ir_if () {} ir_return ir_call()

Note that the basic blocks returned by this don't encompass all operations performed by the program – for example, if conditions don't get returned, nor do the assignments that will be generated for ir_call parameters.
