ir_rvalue* loop_variable::iv_scale

Increment values for loop induction variables.

Loop induction variables have a single increment of the form b * biv + c, where b and c are loop constants and i is a basic loop induction variable.

If iv_scale is NULL, 1 is used. If biv is the same as var, then var is a basic loop induction variable.
