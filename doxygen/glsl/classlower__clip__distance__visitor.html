ir_visitor_status lower_clip_distance_visitor::visit_leave ( ir_call *  ir)
virtual

If gl_ClipDistance appears as an argument in an ir_call expression, replace it with a temporary variable, and make sure the ir_call is preceded and/or followed by assignments that copy the contents of the temporary variable to and/or from gl_ClipDistance.

Each of these assignments is then lowered to refer to gl_ClipDistanceMESA.
