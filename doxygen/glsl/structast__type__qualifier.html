const char * ast_type_qualifier::interpolation_string ( ) const

Return string representation of interpolation qualifier.

If an interpolation qualifier is present, then return that qualifier's string representation. Otherwise, return null. For example, if the noperspective bit is set, then this returns "noperspective".

If multiple interpolation qualifiers are somehow present, then the returned string is undefined but not null.
