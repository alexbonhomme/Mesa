inlinevirtual

If an r-value is a reference to a whole variable, get that variable.

Returns
Pointer to a variable that is completely dereferenced by the r-value. If the r-value is not a dereference or the dereference does not access the entire variable (i.e., it's just one array element, struct field), NULL is returned.
