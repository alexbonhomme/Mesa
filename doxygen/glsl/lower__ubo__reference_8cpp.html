Detailed Description

IR lower pass to replace dereferences of variables in a uniform buffer object with usage of ir_binop_ubo_load expressions, each of which can read data up to the size of a vec4.

This relieves drivers of the responsibility to deal with tricky UBO layout issues like std140 structures and row_major matrices on their own.
