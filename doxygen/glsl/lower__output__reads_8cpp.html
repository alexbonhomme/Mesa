Detailed Description

In GLSL, shader output variables (such as varyings) can be both read and written.

However, on some hardware, reading an output register causes trouble.

This pass creates temporary shadow copies of every (used) shader output, and replaces all accesses to use those instead. It also adds code to the main() function to copy the final values to the actual shader outputs.
