bool ir_constant::is_basis ( ) const
virtual

Determine if an r-value is a basis vector.

The base implementation of this function always returns false. The ir_constant class over-rides this function to return true only for vector and scalar types that have one element set to the value one, and the other elements set to the value zero. For boolean types, the result is always false.
