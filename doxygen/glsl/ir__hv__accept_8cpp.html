bool  statement_list 
)

Process a list of nodes using a hierarchical vistor.

If statement_list is true (the default), this is a list of statements, so v->base_ir will be set to point to each statement just before iterating over it, and restored after iteration is complete. If statement_list is false, this is a list that appears inside a statement (e.g. a parameter list), so v->base_ir will be left alone.

Warning
This function will operate correctly if a node being processed is removed from the list. However, if nodes are added to the list after the node being processed, some of the added nodes may not be processed.
