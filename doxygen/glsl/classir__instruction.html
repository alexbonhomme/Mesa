Constructor & Destructor Documentation

virtual ir_instruction::~ir_instruction ( )
inlinevirtual

GCC 4.7+ and clang warn when deleting an ir_instruction unless there's a virtual destructor present.

Because we almost universally use ralloc for our memory management of ir_instructions, the destructor doesn't need to do any work.
