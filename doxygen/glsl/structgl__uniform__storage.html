uint8_t gl_uniform_storage::index

Base sampler index.

If ::base_type is GLSL_TYPE_SAMPLER, this represents the index of this sampler. If ::array_elements is not zero, the array will use sampler indices ::sampler through ::sampler + ::array_elements - 1, inclusive.

Note that the index may be different in each shader stage.

union gl_constant_value* gl_uniform_storage::storage

Storage used by Mesa for the uniform.

This form of the uniform is used by Mesa's implementation of glGetUniform. It can also be used by drivers to obtain the value of the uniform if the ::driver_storage interface is not used.
